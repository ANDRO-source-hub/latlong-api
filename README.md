# LatLong API


CRUD API untuk menyimpan data Latitude & Longitude.


## Endpoint


- POST /api/locations
- GET /api/locations
- GET /api/locations/:id
- PUT /api/locations/:id
- DELETE /api/locations/:id


## Body POST/PUT


```json
{
"latitude": -6.200000,
"longitude": 106.816666,
"note": "Jakarta"
}
